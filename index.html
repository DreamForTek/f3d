F3D, a fast and minimalist 3D viewer

  • Releases

    F3D (pronounced /f…õd/) supports many file formats, from digital content to scientific datasets (including glTF, stl, ply, obj), can show animations, has a lot of rendering options including real time physically based rendering and Monte Carlo pathtracing.

  • Support Us

    F3D was created at Kitware and is now developed by a team of passionate devs. Please use F3D and star it on github to support us.
